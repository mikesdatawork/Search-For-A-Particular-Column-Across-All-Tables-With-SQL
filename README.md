## About-Process

<p>Here's a quick way to search for a certain column name across all your tables in any database.</p>     


## SQL-Logic
```SQL
use master;
set nocount on
select
  table_name
, column_name
from
information_schema.columns
where
column_name like '%documents%' -- replace documents with whatever you're looking for. order by
table_name
, column_name asc
```
